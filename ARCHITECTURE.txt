【写入流程】

用户调用 InsertRows()
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 1. 获取 worker 令牌（并发控制）                               │
│    workersLimitCh ← struct{}{}                               │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 2. 确保 head 分区是活动的                                     │
│    ensureActiveHead()                                        │
│    ├── 如果 head 不活动 → 创建新内存分区                     │
│    └── 插入到 chunkList 头部                            │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 3. 写入 WAL（预写日志）                                       │
│    wal.append(operationInsert, rows)                         │
│    ├── 编码数据                                              │
│    ├── 写入缓冲区                                            │
│    └── 缓冲区满时刷新到磁盘                                  │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 4. 插入到分区                                                 │
│    chunk.insertRows(rows)                                    │
│    ├── 检查时间戳范围                                        │
│    ├── 过滤过期数据                                          │
│    ├── 生成唯一标识符 (marshalKey)                           │
│    ├── 获取或创建 series                                     │
│    └── 添加数据点到 inOrderPoints 切片                       │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
    返回过时的行（如果时间戳早于分区范围）


【查询流程】

用户调用 Select(metric, labels, start, end)
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 1. 参数验证                                                   │
│    ├── metric 不能为空                                       │
│    └── start < end                                           │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 2. 创建分区迭代器                                             │
│    iterator := chunkList.newIterator()                       │
│    ├── 使用 Dummy Node                                       │
│    └── 从最新分区开始                                        │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 3. 遍历分区                                                   │
│    for iterator.next() {                                     │
│        part := iterator.value()                              │
│        ├── 检查时间范围是否匹配                               │
│        ├── 跳过不相关的分区                                   │
│        └── 从分区查询数据                                    │
│    }                                                          │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 4. 从分区查询数据                                             │
│    part.selectDataPoints(metric, labels, start, end)         │
│    ├── mutableChunk: 遍历 seriesMap，过滤数据点              │
│    └── immutableChunk: mmap 读取，解码数据点                  │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 5. 合并并排序结果                                             │
│    ├── 收集所有分区的结果                                     │
│    └── 按时间戳升序排序                                      │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
    返回 []*DataPoint

触发条件：创建新分区、Close() 调用
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 1. 遍历内存分区（跳过前2个可写分区）                          │
│    for chunk in chunkList[2:] {                             │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 2. 压缩数据                                                   │
│    flush(dirPath, mutableChunk)                             │
│    ├── 创建目录 p-{minTimestamp}-{maxTimestamp}             │

│    ├── 编码所有指标的数据点                                  │
│    ├── 刷新缓冲区                                            │
│    └── 创建 meta.json（分区元数据）                          │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 3. 创建磁盘分区                                               │
│    immutableChunk := openImmutableChunk(dirPath)             │
│    ├── 使用 mmap 映射 data 文件                              │
│    ├── 读取 meta.json                                        │
│    └── 创建 immutableChunk 对象                              │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 4. 替换分区                                                   │
│    chunkList.swap(mutableChunk, immutableChunk)              │
│    └── 在链表中替换节点                                      │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 5. 清理 WAL                                                   │
│    wal.removeOldest()                                        │
│    └── 删除对应的 WAL 段文件                                 │
└─────────────────────────────────────────────────────────────┘


【WAL 恢复流程】

程序启动
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 1. 加载磁盘分区                                               │
│    扫描 data 目录                                            │
│    ├── 打开所有 p-* 目录                                      │
│    ├── mmap 映射 data 文件                                    │
│    └── 读取 meta.json                                        │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│ 2. 恢复 WAL 数据                                             │
│    recoverWAL(walDir)                                        │
│    ├── 读取所有 WAL 段文件                                   │
│    ├── 解码 WAL 记录                                         │
│    ├── 调用 InsertRows() 重新插入数据                         │
│    └── 删除 WAL 文件                                         │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
    创建新的可写分区

╔══════════════════════════════════════════════════════════════════════════════╗
║                              核心设计原则                                       ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. 分区架构（chunk-based）
   ├─ 按时间范围分区（默认1小时）
   ├─ 每个分区独立存储
   ├─ 支持内存分区和磁盘分区
   └─ 最新数据在内存中，历史数据在磁盘

2. 预写日志（WAL）
   ├─ 数据写入前先记录 WAL
   ├─ 防止数据丢失
   └─ 崩溃后可恢复

3. 并发控制
   ├─ 限制并发写入数量（GOMAXPROCS）
   ├─ 读写锁保护分区列表
   ├─ 原子操作计数
   └─ WaitGroup 等待写入完成

4. 内存映射（mmap）
   ├─ 磁盘分区使用 mmap
   ├─ 按需读取，不全部加载
   ├─ 操作系统管理缓存
   └─ 高效访问历史数据

5. 乱序数据处理
   ├─ 保留2个可写分区（最新分区和次新分区）
   ├─ series 结构体支持有序和乱序数据点
   ├─ inOrderPoints 存储有序数据
   ├─ outOfOrderPoints 存储乱序数据
   └─ 丢弃时间戳早于分区最小时间戳的数据

6. 数据压缩
   ├─ 使用位流编码
   ├─ Gorilla 压缩算法
   ├─ 按指标单独压缩
   └─ 减少存储空间


╔══════════════════════════════════════════════════════════════════════════════╗
║                              性能特征                                           ║
╚══════════════════════════════════════════════════════════════════════════════╝

写入性能：
├─ 批量插入优化
├─ 并发写入控制
├─ WAL 缓冲区减少磁盘 I/O
└─ 有序切片提供高缓存命中率

查询性能：
├─ 分区过滤减少扫描范围
├─ mmap 零拷贝读取
├─ 索引快速定位指标
└─ 按需解码数据点

内存占用：
├─ 只有元数据在堆中
├─ 磁盘分区使用 mmap
├─ 乱序数据容量限制
└─ 可配置保留时间

磁盘 I/O：
├─ 批量刷新减少写入次数
├─ 顺序写入优化
├─ 数据压缩减少存储
└─ WAL 刷盘可配置
